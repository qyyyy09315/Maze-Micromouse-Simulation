# 迷宫电脑鼠仿真

## 本地开发

### 环境准备

- 安装 [Node.js](https://nodejs.org/en)
- 安装 [pnpm](https://pnpm.io/installation)

### 操作步骤

- 安装依赖

```sh
pnpm install
```

- 启动 Dev Server

```sh
pnpm run dev
```

- 在浏览器访问 http://localhost:3000

本项目为大连海洋大学《算法设计与分析》课程的期末项目，旨在模拟迷宫电脑鼠的路径规划和导航过程。
This project is a final project for the course "Algorithm Design and Analysis" at Dalian Ocean University. It aims to simulate the path planning and navigation of a computer mouse in a maze.
